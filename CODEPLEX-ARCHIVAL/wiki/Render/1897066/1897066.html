Free Tool for Publishing Third-Party Updates or Applications to Wsus

Now works with Wsus 6.2, 6.3 and 10 (Windows Server 2012, 2012R2, and 2016)

Project Description :
English :
Wsus Package Publisher allows you to publish your own updates as MSI, MSP or EXE files. Hence, you can deploy applications like Adobe Reader, Java, Flash Player or Symantec Endpoint Protection 12.1. and update them.

You can import updates from vendor catalogs like Dell, HP and Fujitsu to publish hardwareupdates like drivers or new BIOS versions to your servers and PCs.

You must have a WSUS infrastructure installed and running to use this application. With this application you can:
- Publish
- Approve
- Delete
- Review
... updates.
You can also generate reports to track the status of installation of updates.
You can even find computers that are in AD but not in Wsus.
